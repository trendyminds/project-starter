Sprites

Sprites are automatically generated for you by the SVGs you drop in the icons directory. Use sprites in a :before or :after... or right in the DOM!

Generated by svg-sprite

Use sprites as often as possible to reduce the number of HTTP requests the browser needs to make to the server.

Dynamically embedded inline SVGs

Occasionally you might need to animate or colorize an icon normally used in the spritesheet. If so, you may dynamically pull in the SVG code from the icon and use CSS for styling.
